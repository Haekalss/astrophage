# Astrophage
Astrophage merupakan mikroba alien energi bintang yang berkembang biak dengan cepat dan mengancam kehidupan di Bumi dengan meredupkan Matahari. 

Gimmick ini diambil dari salah satu scene di Film Project Hail Mary.

Cara pakai:
1. Izinkan akses kamera saat diminta.
2. Tutup atau redupkan kamera untuk melihat perubahan tampilan astrophage (titik-titik merah).
3. Drag atau geser layar untuk memutar tampilan.
